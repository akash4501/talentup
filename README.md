## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/TalentUp.git
```

Go to project folder

```bash
cd TalentUp
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
